# Front in Floripa 2018
> Front in Floripa's website

# Como rodar o projeto

1 - Instale o Gulp globalmente

```
[sudo] npm install -g gulp
```

2 - Instale as dependências

```
npm install
```

3 - Rode o projeto

```
gulp server
```

OBS.: Se quiser só dar build, apenas rode:

```
gulp build
```
